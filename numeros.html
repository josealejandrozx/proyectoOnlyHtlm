Glosario rápido

  • Número: símbolo para contar o medir.
  • Fracción: parte de un todo (a/b).
  • Decimal: forma con punto (0.5).

Curiosidad

Algunos matemáticos incluyen el 0 como número natural; otros no. Aquí te explicamos ambas versiones.

¿Qué es un número? (explicación para principiantes)

Un número es un símbolo que usamos para contar, medir y comparar cantidades. Por ejemplo: si tienes 3 manzanas, el número 3 representa cuántas manzanas hay.

Números naturales (N)

Son los que usamos para contar: 1, 2, 3, 4, ... En algunos libros se incluye el 0 como natural.

Números enteros (Z)

Incluyen positivos, negativos y el cero: ..., -3, -2, -1, 0, 1, 2, 3, ....

Ejemplo: la temperatura puede ser -2 °C.

Números racionales (Q)

Se escriben como fracción a/b. Ejemplos: 1/2, 3/4, 5/1.

Números decimales

Usan un punto: 2.5, 0.75. Algunos terminan, otros se repiten o son infinitos.

Números irracionales y reales

No se pueden expresar como fracción. Ejemplos: √2, π. Los números reales incluyen racionales e irracionales.

Usos prácticos

Los números sirven para: contar dinero, medir distancias, leer relojes, cocinar y mucho más.

Ejercicios

  1. Cuenta: 4 lápices + 3 lápices = ?
  2. Clasifica: 5, -2, 1/4, π.
  3. Convierte: 3/5 a decimal.
  4. Ubica en la recta: -2, 0, 1/2, 2.

Soluciones

  • 7 lápices.
  • 5 = natural; -2 = entero; 1/4 = racional; π = irracional.
  • 3 ÷ 5 = 0.6.
  • Recta: -2 (izq), 0 (centro), 1/2 (entre 0 y 1), 2 (derecha).

Errores comunes

  • Confundir enteros con naturales.
  • Olvidar que hay decimales infinitos.
